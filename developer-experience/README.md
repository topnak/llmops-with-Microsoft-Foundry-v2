# Developer Experience — LLMOps Hands-on Lab

Welcome! This folder contains everything you need to **experience the LLMOps
development loop** as a developer working on an AI Agent using Microsoft Foundry.

## What You Will Do

You will act as an **LLMOps engineer** who:
1. Sets up a local development environment
2. Edits and improves prompts for a retail AI agent
3. Runs tests, evaluations, and model comparisons
4. Commits changes that trigger automated CI/CD pipelines
5. Reviews pipeline results in GitHub Actions

## Prerequisites

- **Python 3.11+** installed
- **Git** installed
- **VS Code** with Python extension
- **Azure CLI** (`az`) installed — for service principal setup
- Access to the **Microsoft Foundry** project (ask your admin for credentials)
